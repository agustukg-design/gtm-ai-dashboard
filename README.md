# 🚀 GTM-AI Aggregator Dashboard (Solana Ecosystem)

An all-in-one AI-powered aggregator dashboard that synthesizes on-chain analytics, market sentiment, and predictive data from multiple global platforms into a single unified control panel. This project utilizes the `$GTM` token on the Solana network as its core utility for premium feature access.

## 🛠️ Key Features & API Integrations

The dashboard is designed to integrate 5+ major APIs in real-time to deliver actionable trading intelligence:
* **GMGN.ai API:** Monitors new token launches on Solana, tracks "Smart Money" movements, and analyzes insider wallet distributions.
* **Polymarket API:** Extracts global sentiment and betting volume data regarding political events, crypto trends, and real-world outcomes.
* **Social Sentiment API (X/Twitter):** Analyzes viral memecoin keyword trends and social velocity over the last 24 hours.
* **CoinGecko / Birdeye API:** Provides accurate real-time token price charts, liquidity metrics, and trading volumes.
* **AI Core Processor (OpenAI/Anthropic):** Processes and synthesizes all raw data points into concise, AI-generated insights and trading signals for users.

## 🪙 $GTM Token Utility
The `$GTM` token serves as the exclusive native utility currency on the platform used for:
1. Subscribing to advanced analytical features (Premium Tier Dashboard).
2. Unlocking real-time smart wallet and whale tracking tools (via GMGN data integration).
3. Accessing automated AI-driven market prediction models.
